# OpenNNA2.0
OpenNNA2.0-更易用，更通用，更多算子......

### OpenNNA核心特性

1. 适用性，移植性: 相同代码，可用于stm32，fpga，pc平台
2. 对于fpga:多加速器(NNA)调度，负载均衡
3. 提供算子开发框架，可自己开发自己的算子。为了独立性，一个算子一个.c文件
4. 提供demo: 1.手写数字 2.KWS(关键词识别) 3.yolo v1目标检测
